# 需求历史

## 2026-04-16: 离职人员自动交接
- 状态：已完成全流程分析
- 摘要：员工标记离职后，系统自动把负责的项目和产品（PO）交接给直属上级
- 关键决策：
  - 触发方式：每天凌晨批量处理
  - 默认接收人：直属上级
  - 接收方式：上级全部接收或逐条转派给他人
  - 转派范围：全公司可选
  - 超时提醒：3天未处理，通知上级+系统管理员
  - 不需要灰度，全量上线
- 涉及模块：人事服务、项目管理、产品管理、组织架构、通知服务
- 边界场景：
  - 上级也离职 → 递归向上找
  - 找不到上级 → 通知管理员
  - 上级是自己 → 数据异常，不处理
  - 项目已关闭 → 不处理
  - 项目有多个负责人 → 只移除离职人
  - 批量离职(100人) → 性能保障
  - 任务中断 → 事务回滚，下次重试
- 验收标准：9项（已生成完整 checklist）
